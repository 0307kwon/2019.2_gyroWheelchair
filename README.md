2019.2_gyroWheelchair
========================

## 1. summary

2019년 2학기에 개발한 자이로 휠체어

### - 자이로 휠체어의 기능

1) 컬러인식센서로 휠체어의 전진,정지,후진을 구현

2) 모터의 엔코더를 이용해 휠체어의 회전을 구현

3) 조종기와 차체와의 통신 방법으로 블루투스를 사용, 무선 조종이 가능함

4) 차체가 한쪽으로 기울면 휠체어의 의자는 반대쪽으로 기울게하여 항상 같은 자세를 유지하도록 구현
